# 句型英语

> 用一个句型，解锁一百种表达

## 产品简介

**句型英语**是一款面向中文用户的英语学习工具，核心理念是「以句型为纲，以例句为目」。我们相信，掌握一个核心句型，就能衍生出无数实用表达，比死记硬背单词更高效。

## 核心功能

### 📚 句型学习
精选日常生活高频句型，从自我介绍到情感表达，覆盖真实交流场景。每个句型配备 5+ 实用例句，即学即用。

### 🔍 智能搜索
支持中英双语搜索，输入关键词即可找到相关句型和例句。不知道怎么说？搜一下就有答案。

### 📊 学习追踪
自动记录学习进度，随时查看已掌握和待学习的句型，让进步看得见。

### ⚡ 碎片化学习
移动端优先设计，通勤、排队、睡前，随时随地打开就能学。

## 适用人群

- **英语初学者** — 建立正确的句子结构意识
- **备考学生** — 积累口语和写作素材
- **职场人士** — 提升日常英语表达能力
- **自学者** — 系统化的句型学习路径

## 快速开始

访问网站，选择一个句型，开始你的第一句：

> **I'm a ____**  
> 我是一名……  
> *I'm a teacher. / I'm a developer. / I'm a dreamer.*
